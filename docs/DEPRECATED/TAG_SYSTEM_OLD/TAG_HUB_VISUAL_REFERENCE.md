## Color Scheme

The Tag Hub uses the following color scheme:

- **Background**: Dark glass morphism (`bg-background`)
- **Borders**: White with 10% opacity (`border-white/10`)
- **Text**: White for primary, gray shades for secondary
- **Accents**:
  - Blue (`blue-400/500`) for Browse and primary actions
  - Green (`green-400/500`) for "keep" tags in Fusion
  - Red (`red-400/500`) for "delete" tags in Fusion
  - Purple (`purple-400/500`) for AI tags and merge operations
- **Hover States**: Subtle color shifts with transition animations

## Responsive Design

The Tag Hub is optimized for desktop use with a maximum width of 5xl (80rem). 
It features:
- Responsive padding and spacing
- Flexible grid layouts that adapt to content
- Smooth animations using Framer Motion
- Keyboard-first navigation with visual feedback

## Accessibility Features

- **ARIA Labels**: All interactive elements have proper labels
- **Keyboard Navigation**: Full keyboard support for all operations
- **Focus Indicators**: Clear visual feedback for focused elements
- **Screen Reader Support**: Semantic HTML and proper role attributes
- **Color Contrast**: All text meets WCAG AA standards

## Animation Details

- **Modal Entrance**: Fade in with scale animation (0.95 to 1.0)
- **Tab Switching**: Content fades in smoothly
- **Hover Effects**: Subtle scale and color transitions
- **Loading States**: Spinning refresh icon
- **Button Clicks**: Active scale animation (0.95)

## Dimensions

- **Modal Width**: 80rem (1280px) max
- **Modal Height**: 85vh max
- **Grid Columns**: 2-4 depending on screen size
- **Padding**: 1.5rem (24px) standard
- **Border Radius**: 0.75rem (12px) for cards
